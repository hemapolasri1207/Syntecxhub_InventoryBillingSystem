# 🧾 Inventory Billing System (Java)

A simple console-based Inventory Billing System developed using Java.  
This project helps in managing inventory items and generating bills efficiently using collections.

---

## 🚀 Features

- 📦 Add new items to inventory
- 📋 Display available items
- 🔄 Update stock quantities
- 🧾 Generate bill for selected items
- 💰 Automatic total calculation
